# Lua API 

## ChangePicture(Param1, Param2)

    - 픽처의 화면전환을 수행하는 함수입니다.

<br>

### Parameter (인자)

    - Param1 : 켄버스 이름
    - Param2 : 픽처 이름

### Return (반환값)

	- none

### Remark (주의사항)
    - none

### Example (함수사용 예시)

```lua
	function _onclick()
		ChangePicture(＂this＂, ＂picture\\ocean.svg＂)
	end
```
