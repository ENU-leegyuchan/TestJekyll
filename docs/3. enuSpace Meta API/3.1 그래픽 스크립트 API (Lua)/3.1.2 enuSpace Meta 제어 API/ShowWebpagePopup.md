# Lua API 

## ShowWebpagePopup(Param1)

    - Web page 주소를 읽어 새로운 창에 화면을 호출하는 함수입니다.

<br>

### Parameter (인자)

    - Param1 : path

### Return (반환값)

	- none

### Remark (주의사항)
    - none

### Example (함수사용 예시)

```lua
	function _onclick()
        ShowWebpagePopup("https://www.youtube.com/embed/4dwpcSRX-1s") 
	end
```
