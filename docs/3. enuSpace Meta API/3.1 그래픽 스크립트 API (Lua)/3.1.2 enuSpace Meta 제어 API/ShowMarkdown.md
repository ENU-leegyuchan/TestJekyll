# Lua API 

## ShowMarkdown(Param1)

    - MarkDown 파일을 읽어와 표시하는 함수입니다.

<br>

### Parameter (인자)

    - Param1 : path

### Return (반환값)

	- none

### Remark (주의사항)
    - none

### Example (함수사용 예시)

```lua
	function _onclick()
        ShowMarkdown("document/document.md")
	end
```
